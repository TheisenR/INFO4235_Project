# INFO4235_Project
A student marketplace web application made using Meteor, JavaScript, HTML5, CSS3, and MongoDB .

## Project layout (reorganized)

- src/
	- client/  — React entry and components (App.jsx, main.jsx, pages)
	- server/  — server code and API backups
- public/   — static HTML served in development/production
- config/   — tooling/config files (vite.config.js)
- scripts/  — utility scripts and playgrounds

To run the original Meteor app keep using `meteor run`. For a Vite-based frontend development workflow, run `npm run dev` after installing dev dependencies.
